# Data Workbench Cookie{#data-workbench-cookies}

Data Workbench 會使用 Cookie 來即時分析迅速變遷的大量資料。

**Cookie 名稱：v1st**

| 屬性 | 說明 |
|---|---|
| 儲存的資訊 | 不重複訪客 ID 時間/日期戳記 |
| 過期 | 此 cookie 為永久性。 |
| 使用方式 | 該 Cookie 用於識別不重複訪客。 |
| 位置 | 此 Cookie 儲存在頁面的網域中。 |
| 大小 | 8 位元組，若包含「name =」部分 (v1st=)，則為 13 位元組 |
