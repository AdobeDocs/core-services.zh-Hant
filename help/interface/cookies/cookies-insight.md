# Data Workbench Cookie{#data-workbench-cookies}

Data Workbench 使用 Cookie 來即時分析迅速變遷的大量資料。

**Cookie 名稱: v1st**

| 屬性 | 說明 |
|---|---|
| 儲存資訊 | 獨特訪客 ID 時間/日期戳記 |
| 過期 | 此乃永久性 Cookie |
| 使用狀況 | 該 Cookie 用於識別獨特訪客   。 |
| 位置 | 該 Cookie 儲存在頁面的網域上。 |
| 大小 | 8 位元組；如果包含 "name =" 部分 (v1st=) 則為 13 位元組 |
