# Data Workbench Cookie{#data-workbench-cookies}

Data Workbench 使用 Cookie 來即時分析迅速變遷的大量資料。

**Cookie 名稱: v1st**

| 屬性 | 說明 |
|---|---|
| 儲存的資訊 | 獨特訪客ID時間／日期戳記 |
| 過期 | 這是永久性Cookie |
| 使用狀況 | 該 Cookie 用於識別獨特訪客    。 |
| 位置 | 此Cookie儲存在頁面的網域。 |
| 大小 | 8個位元組，若包含&quot;name =&quot;部分(v1st=)，則為13個位元組 |
