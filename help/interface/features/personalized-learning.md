# 個人化學習推薦和促銷

Adobe會根據您在Adobe產品中的工作，為您提供量身打造的實用內容。 產品使用資料會告知我們如何為您量身打造此內容。 您可以在[Experience Cloud偏好設定](https://experience.adobe.com/preferences)頁面上選擇加入或選擇退出分享您的產品使用資料。
